# 변경 이력

## [2.0.0] - 2025-11-21

### 🎨 카카오모빌리티 브랜드 리브랜딩 (Phase 1 완료)

#### ✨ 주요 변경사항

**브랜드 컬러 시스템 적용**
- 카카오모빌리티 Primary: `#FF5722` (오렌지)
- 카카오모빌리티 Secondary: `#FFC107` (노란색)
- 카카오 그라데이션: `linear-gradient(135deg, #FF5722 0%, #FF9800 100%)`
- 기존 보라색 그라데이션 완전 제거

**UI/UX 개선**
- 헤더에 카카오모빌리티 로고(🚕) 및 브랜드 텍스트 추가
- 모든 버튼, 카드, 배지를 카카오 브랜드 컬러로 통일
- 네비게이션 버튼 스타일 개선 (더 부드러운 전환 효과)
- 모달 디자인 개선 및 정보 표시 방식 향상

**사용자 경험 개선**
- `alert()` 대신 `react-toastify`를 사용한 토스트 알림 시스템
- `framer-motion`을 사용한 부드러운 애니메이션 효과
- 모달 오픈/클로즈 시 스프링 애니메이션 적용
- 차량 정보에 상세 스펙 추가 (전기차 타입, 주행거리, 좌석수 등)

**반응형 디자인 강화**
- 모바일 우선 디자인 적용
- 480px, 768px, 1024px 브레이크포인트 최적화
- 터치 스크린 환경 고려한 버튼 크기 조정

**접근성(Accessibility) 개선**
- `focus-visible` 스타일 추가로 키보드 네비게이션 개선
- `prefers-reduced-motion` 지원으로 애니메이션 최적화
- 스크린 리더를 위한 `.sr-only` 클래스 추가

**성능 최적화**
- CSS 변수를 사용한 컬러 시스템으로 테마 변경 용이
- 커스텀 스크롤바 디자인 (브랜드 컬러 적용)
- 로딩 스피너 추가 (향후 비동기 작업에 활용)

#### 📦 Dependencies 추가
```json
{
  "framer-motion": "^11.x",
  "react-toastify": "^10.x"
}
```
