# 🚕 카카오모빌리티 차량 대여 시스템

카카오모빌리티 직원들을 위한 차량 대여 관리 웹 애플리케이션입니다.

## 📋 프로젝트 개요

관리자가 대여 기간을 설정하고, 직원들이 차량을 신청하며, 실시간으로 경쟁률을 확인할 수 있는 시스템입니다.

### 대여 가능 차량
- 🏎️ 포르쉐 타이칸 4S (전기차, 주행거리 407km, 4인승)
- 🚗 벤츠 EQS 450+ (전기차, 주행거리 625km, 5인승)

### 대여 시간대
주당 2회로 구분하여 운영됩니다:

1. **1회차**: 월요일 오후 6시 ~ 목요일 오후 6시 (시간 엄수)
2. **2회차**: 금요일 오전 10시 ~ 월요일 오전 10시 (시간 엄수)

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하면 애플리케이션을 확인할 수 있습니다.

### 빌드

```bash
npm run build
```

## ✨ 주요 기능

### 1. 관리자 페이지 ⚙️
- 대여 시작 날짜와 종료 날짜 설정
- 📅 **추천 날짜 시스템**: 1회차/2회차가 완전하게 포함되는 최적의 날짜 자동 제안
- 설정된 기간 확인
- 해당 기간의 달력이 자동으로 생성됨

### 2. 대여 신청 페이지 📅
- 달력 뷰에서 직접 원하는 날짜를 클릭하여 신청
- 차량별, 회차별 필터링 기능
- 실시간 경쟁률 확인 (예: 3:1)
- 신청자 미리보기
- **모달 기반 신청 시스템**:
  - 한글 이름 입력
  - 영어 아이디 입력
  - 선택한 차량/회차/날짜 자동 표시
- 중복 신청 방지 기능

### 3. 달력 보기 🗓️
- 월별 캘린더 뷰
- 각 차량별, 회차별 신청 현황 표시
- **시각적 표시**:
  - 🟢 시작일: 클릭하여 신청 가능
  - ➡️ 대여 기간: 화살표로 표시
  - 🔵 종료일: "종료" 마크 표시
  - ⚪ 불완전한 회차: 회색 처리
- 경쟁률 배지 실시간 업데이트
- 신청자 목록 미리보기 (최대 3명 표시)

### 4. 사용자 경험 개선
- 🎉 **토스트 알림**: `react-toastify`를 사용한 친화적인 알림 시스템
- ✨ **부드러운 애니메이션**: `framer-motion`을 사용한 모달 및 페이지 전환
- 📱 **완벽한 반응형**: 모바일, 태블릿, 데스크톱 모든 환경 최적화
- ♿ **접근성 강화**: 키보드 네비게이션, 스크린 리더 지원

### 5. 데이터 저장
- 로컬 스토리지를 이용한 데이터 영구 저장
- 새로고침해도 데이터 유지

## 🎯 사용 흐름

1. **관리자**: "⚙️ 관리자" 버튼 클릭 → 시작/종료 날짜 설정 (추천 날짜 활용)
2. **사용자**: "📅 대여 신청" 버튼 클릭 → 차량/회차 선택 → 달력에서 날짜 클릭 → 정보 입력
3. **확인**: 달력에서 실시간 경쟁률 및 신청자 목록 확인

## 🎨 기술 스택

- **Frontend**: React 19.2.0
- **Animation**: Framer Motion
- **Notifications**: React Toastify
- **Styling**: CSS3 (카카오모빌리티 브랜드 컬러 시스템)
- **Storage**: Local Storage
- **Testing**: Jest, React Testing Library

### 브랜드 컬러 시스템
```css
--kakao-primary: #FF5722 (오렌지)
--kakao-secondary: #FFC107 (노란색)
--kakao-gradient: linear-gradient(135deg, #FF5722 0%, #FF9800 100%)
```

## 📱 반응형 디자인

- **데스크톱**: 1400px+ (최적 경험)
- **태블릿**: 768px - 1024px
- **모바일**: 320px - 767px

## 🔒 주요 규칙

- ✅ 관리자가 먼저 대여 기간을 설정해야 함
- ✅ 각 사용자는 동일한 차량/회차에 한 번만 신청 가능
- ✅ 1회차/2회차가 완전하지 않으면 신청 불가 (자동 필터링)
- ✅ 시간 엄수 필수
- ✅ 경쟁률이 1:1 이상이면 추첨 필요
